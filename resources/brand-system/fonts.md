# Polices

- Display: Cormorant Garamond
- Body: Plus Jakarta Sans

## Usage

- `Cormorant Garamond` pour le hero, les noms de parfums, les titres editoriaux et les grands messages
- `Plus Jakarta Sans` pour la navigation, les paragraphes, les labels et les informations de formulaire

## Intention

- Le display doit paraitre raffine, fluide et sensoriel
- Le body doit rester net, actuel et facile a lire
