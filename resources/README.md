# Ressources Projet

Ce dossier regroupe les ressources de reference pour reconstruire l'application `Maison Parfums Store` en francais.
Il suit la progression du cours: frontend d'abord, backend ensuite, puis connexion frontend/backend a la fin.

## A lire d'abord

- [START-HERE.md](./START-HERE.md)
- [CHECKLIST.md](./CHECKLIST.md)
- [COMMON-MISTAKES.md](./COMMON-MISTAKES.md)
- [ui-guide/ui-guide.html](./ui-guide/ui-guide.html)

## Contenu

- `brand-system/`: regles visuelles de la niche parfums
- `content/`: textes de l'interface
- `seed-data/`: donnees produits a utiliser plus tard pour MongoDB et l'API
- `ui-guide/`: guide visuel HTML et assets placeholders

## Positionnement de la niche

- univers: haute parfumerie
- ton: sensoriel, editorial, premium
- produits: flacons, collections signatures, cadeaux premium
