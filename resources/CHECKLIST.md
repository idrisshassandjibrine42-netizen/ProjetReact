# Checklist

Utiliser cette checklist dans l'ordre du cours.

## Phase 1. HTML

- creer les pages principales
- ajouter le header et le footer
- faire la homepage
- faire la page produits
- faire la page detail produit
- faire la page panier
- faire la page checkout
- faire la page contact
- faire la page mentions legales

## Phase 2. CSS

- fond creme clair
- navbar sombre
- bonne hierarchie de titres
- carousel visuellement simple sous le hero
- cartes produit propres
- footer sombre avec accents clairs
- responsive mobile correct

## Phase 3. JavaScript

- menu hamburger mobile
- panier fonctionnel
- carousel scrollable
- clic produit -> detail produit
- validation visible sur le checkout

## Phase 4. React avec Vite

- transformer chaque page en composant
- creer les composants reutilisables
- separer layout, pages et composants
- utiliser le routing proprement

## Phase 5. Tailwind CSS

- appliquer les espacements finaux
- reproduire la palette du theme
- reproduire la typographie du site
- finaliser le responsive

## Phase 6. Backend avec Node.js, Express, MongoDB

- creer l'API produits
- connecter MongoDB
- injecter les produits depuis `seed-data/products.fr.json`
- servir les visuels si necessaire

## Phase 7. Connexion frontend/backend

- recuperer les produits depuis l'API
- afficher la liste produits
- afficher la fiche detail
- utiliser les vraies donnees dans le carousel et les pages

## Verification finale

- textes conformes a `content/pages/`
- contact tunisien
- footer avec `Mentions legales`
- carousel photo + nom seulement
- description visible seulement sur la page detail produit
- checkout avec ville en liste et champs obligatoires visibles
